# Contributing to GitChess ♟️

Thank you for your interest in contributing to **GitChess**! We welcome contributions from developers of all skill levels.

---

## 🛠️ Getting Started

### 1. Fork and Clone
Fork the repository on GitHub and clone your fork locally:
```bash
git clone https://github.com/YOUR-USERNAME/git-chess.git
cd git-chess
```

### 2. Set Up Environment
Install dependencies in editable mode:
```bash
pip install -e .
pip install pytest
```

### 3. Initialize Git Hooks
Enable local move validation hooks:
```bash
git-chess init
```

---

## 🧪 Running Tests

Ensure all unit tests pass before submitting a Pull Request:
```bash
python3 -m pytest
```

---

## 📋 Pull Request Process

1. Create a feature branch: `git checkout -b feature/amazing-feature`
2. Make your changes and commit them following clear commit messages.
3. Ensure all tests pass: `python3 -m pytest`
4. Push to your branch and open a Pull Request against `master`.

---

## 💡 How to Help

Check out the [Open Issues](https://github.com/004Ongoro/git-chess/issues) to find good first issues to work on!
